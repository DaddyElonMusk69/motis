Vibe Trading has a sophisticated backtesting system with two key components:

2. **Backtest Engine** - Market-specific execution engines that simulate trading with realistic market rules

## Part 1: Signal Engines

### What is a Signal Engine?

A signal engine is a **Python class** that implements a trading strategy's signal generation logic **without any LLM reasoning**. It's purely deterministic code.

### Example: SMC Signal Engine

The Smart Money Concepts signal engine demonstrates the pattern:

```python
class SignalEngine:
    def __init__(self, swing_length: int = 10, close_break: bool = True):
        self.swing_length = swing_length
        self.close_break = close_break
    
    def generate(self, data_map: Dict[str, pd.DataFrame]) -> Dict[str, pd.Series]:
        result = {}
        for code, df in data_map.items():
            signal = pd.Series(0, index=df.index)
            
            # 1. Detect swing highs/lows
            swing_hl = smc.swing_highs_lows(df, swing_length=self.swing_length)
            
            # 2. Detect BOS/ChoCH (structure breaks)
            bos_choch = smc.bos_choch(df, swing_highs_lows=swing_hl, 
                                      close_break=self.close_break)
            
            # 3. Detect FVG (fair value gaps)
            fvg = smc.fvg(df)
            
            # 4. Combine signals
            structure = bos_choch["CHOCH"].fillna(bos_choch["BOS"]).fillna(0)
            fvg_val = fvg["FVG"].fillna(0)
            
            # 5. Generate final signal (structure + FVG filter)
            buy = (structure == 1) & (fvg_val >= 0)
            sell = (structure == -1) & (fvg_val <= 0)
            signal = buy.astype(int) - sell.astype(int)
            
            result[code] = signal
        return result
```

### Signal Engine Characteristics

**Deterministic:**
- Same input → same output (no randomness, no LLM calls)
- Fully reproducible backtests

**Vectorized:**
- Operates on entire DataFrames at once
- Uses pandas/numpy for performance

**Stateless:**
- No memory between calls
- All state is in the DataFrame index

**Signal Convention:**
- `1` = Long (buy signal)
- `-1` = Short (sell signal)
- `0` = Neutral (no position)

### Where Signal Engines Live

In Vibe Trading, signal engines are stored in two places:

1. **Example implementations:** `skills/<skill-name>/example_signal_engine.py`
   - These are reference implementations
   - Used for testing and documentation
   - 14 skills have example signal engines

2. **User-written strategies:** `<run_dir>/code/signal_engine.py`
   - User writes their own `SignalEngine` class
   - Loaded dynamically by the backtest runner
   - Can be generated by the agent from conversation

#### 3. Signal Alignment

**Critical step:** Signals are shifted by 1 bar to simulate next-bar-open execution:

```python
def _align(data_map, signal_map, codes, optimizer=None):
    """Align signals to dates and shift for next-bar execution."""
    
    # 1. Build unified date index
    all_dates = set()
    for df in data_map.values():
        all_dates.update(df.index)
    dates = sorted(all_dates)
    
    # 2. Build close price matrix
    close_df = pd.DataFrame(index=dates, columns=codes)
    for code in codes:
        close_df[code] = data_map[code]["close"].reindex(dates)
    close_df = close_df.ffill().bfill()  # Forward/backward fill gaps
    
    # 3. Build target position matrix
    pos_df = pd.DataFrame(0.0, index=dates, columns=codes)
    for code in codes:
        raw_signal = signal_map[code].reindex(dates).fillna(0.0)
        # CRITICAL: Shift by 1 bar (signal at T → execute at T+1 open)
        pos_df[code] = raw_signal.shift(1).fillna(0.0)
    
    # 4. Normalize weights (sum of absolute weights <= 1.0)
    scale = pos_df.abs().sum(axis=1).clip(lower=1.0)
    pos_df = pos_df.div(scale, axis=0)
    
    # 5. Optional: apply portfolio optimizer
    if optimizer:
        ret_df = close_df.pct_change().fillna(0.0)
        pos_df = optimizer(ret_df, pos_df, dates)
    
    return dates, close_df, pos_df, ret_df
```

**Why shift by 1 bar?**
- Signal generated at bar T uses data up to bar T close
- Can't execute at bar T close (already happened)
- Execute at bar T+1 open (realistic)

#### 4. Market Engines

Each market has different rules. Engines enforce these rules:

**BaseEngine (abstract):**
```python
class BaseEngine(ABC):
    @abstractmethod
    def can_execute(self, symbol, direction, bar) -> bool:
        """Can this trade execute? (e.g., A-shares can't short)"""
    
    @abstractmethod
    def round_size(self, raw_size, price) -> float:
        """Round to lot size (e.g., crypto: 0.001 BTC minimum)"""
    
    @abstractmethod
    def calc_commission(self, size, price, direction, is_open) -> float:
        """Calculate fees (e.g., maker/taker, stamp tax)"""
    
    @abstractmethod
    def apply_slippage(self, price, direction) -> float:
        """Apply slippage (e.g., 0.1% for market orders)"""
    
    def on_bar(self, symbol, bar, timestamp):
        """Per-bar hooks (e.g., funding fees, liquidation checks)"""
```

**CryptoEngine:**
```python
class CryptoEngine(BaseEngine):
    def can_execute(self, symbol, direction, bar):
        return True  # Crypto allows long and short
    
    def round_size(self, raw_size, price):
        # Round to 0.001 BTC minimum
        return round(raw_size, 3)
    
    def calc_commission(self, size, price, direction, is_open):
        # Maker: 0.02%, Taker: 0.05%
        rate = 0.0002 if is_open else 0.0005
        return size * price * rate
    
    def apply_slippage(self, price, direction):
        # 0.1% slippage
        return price * (1 + direction * 0.001)
    
    def on_bar(self, symbol, bar, timestamp):
        # Apply funding fee for perpetual futures
        if "funding_rate" in bar:
            pos = self.positions.get(symbol)
            if pos:
                funding = pos.size * bar["close"] * bar["funding_rate"]
                self.capital -= funding
```

**ChinaAEngine:**
```python
class ChinaAEngine(BaseEngine):
    def can_execute(self, symbol, direction, bar):
        # A-shares: no shorting, limit up/down blocks trading
        if direction == -1:
            return False  # No short selling
        if bar.get("limit_up") or bar.get("limit_down"):
            return False  # Can't trade at limit
        return True
    
    def round_size(self, raw_size, price):
        # Round to 100 shares (1 lot)
        return round(raw_size / 100) * 100
    
    def calc_commission(self, size, price, direction, is_open):
        # Commission: 0.03%, Stamp tax: 0.1% (sell only)
        comm = size * price * 0.0003
        if not is_open:  # Selling
            comm += size * price * 0.001  # Stamp tax
        return comm
    
    def apply_slippage(self, price, direction):
        # 0.05% slippage
        return price * (1 + direction * 0.0005)
```

#### 5. Bar-by-Bar Execution

```python
def _execute_bars(self, dates, data_map, close_df, target_pos, codes):
    """Execute backtest bar by bar."""
    
    for i, timestamp in enumerate(dates):
        self._bar_idx = i
        
        # Step 1: Per-bar hooks (funding fees, liquidation checks)
        for symbol in codes:
            if timestamp in data_map[symbol].index:
                bar = data_map[symbol].loc[timestamp]
                self.on_bar(symbol, bar, timestamp)
        
        # Step 2: Rebalance each symbol to target weight
        equity = self._calc_equity(close_df, timestamp)
        
        for symbol in codes:
            target_weight = target_pos.at[timestamp, symbol]
            self._rebalance(symbol, target_weight, data_map[symbol], 
                          timestamp, equity)
        
        # Step 3: Record equity snapshot
        snap_equity = self._calc_equity(close_df, timestamp)
        unrealized_pnl = sum(
            pos.direction * pos.size * (current_price - pos.entry_price)
            for pos in self.positions.values()
        )
        self.equity_snapshots.append(EquitySnapshot(
            timestamp=timestamp,
            capital=self.capital,
            unrealized=unrealized_pnl,
            equity=snap_equity,
            positions=len(self.positions)
        ))
    
    # Step 4: Force close all positions at end
    last_timestamp = dates[-1]
    for symbol in list(self.positions.keys()):
        price = close_df.at[last_timestamp, symbol]
        self._close_position(symbol, price, last_timestamp, "end_of_backtest")
```

#### 6. Rebalancing Logic

```python
def _rebalance(self, symbol, target_weight, df, timestamp, equity):
    """Adjust position to match target weight."""
    
    # Determine target direction
    target_dir = 1 if target_weight > 0.01 else (-1 if target_weight < -0.01 else 0)
    current_pos = self.positions.get(symbol)
    
    # Get current bar
    if timestamp not in df.index:
        return
    bar = df.loc[timestamp]
    
    # Step 1: Close if needed (direction change or target is flat)
    if current_pos and (target_dir == 0 or target_dir != current_pos.direction):
        if not self.can_execute(symbol, 0, bar):
            return  # Can't close (e.g., limit down)
        
        open_price = bar["open"]
        exit_price = self.apply_slippage(open_price, -current_pos.direction)
        self._close_position(symbol, exit_price, timestamp, "signal")
    
    # Step 2: Open new position if target is non-zero
    if target_dir != 0 and symbol not in self.positions:
        if not self.can_execute(symbol, target_dir, bar):
            return  # Can't open (e.g., A-share can't short)
        
        open_price = bar["open"]
        entry_price = self.apply_slippage(open_price, target_dir)
        
        # Calculate position size
        leverage = self.default_leverage
        target_notional = abs(target_weight) * equity * leverage
        raw_size = target_notional / entry_price
        size = self.round_size(raw_size, entry_price)
        
        if size <= 0:
            return
        
        # Calculate margin and commission
        margin = size * entry_price / leverage
        commission = self.calc_commission(size, entry_price, target_dir, is_open=True)
        
        # Check if we have enough capital
        if margin + commission > self.capital:
            # Reduce size to fit available capital
            available = self.capital - commission
            if available <= 0:
                return
            size = self.round_size(available * leverage / entry_price, entry_price)
            margin = size * entry_price / leverage
            commission = self.calc_commission(size, entry_price, target_dir, is_open=True)
        
        # Deduct margin and commission from capital
        self.capital -= (margin + commission)
        
        # Open position
        self.positions[symbol] = Position(
            symbol=symbol,
            direction=target_dir,
            entry_price=entry_price,
            entry_time=timestamp,
            size=size,
            leverage=leverage,
            entry_bar_idx=self._bar_idx,
            entry_commission=commission
        )
```

#### 7. Metrics Calculation

```python
def calc_metrics(equity_series, trades, initial_capital, bars_per_year, bench_ret):
    """Calculate comprehensive backtest metrics."""
    
    # Returns
    returns = equity_series.pct_change().fillna(0.0)
    total_return = (equity_series.iloc[-1] / initial_capital - 1) * 100
    annual_return = ((equity_series.iloc[-1] / initial_capital) ** 
                    (bars_per_year / len(equity_series)) - 1) * 100
    
    # Risk metrics
    volatility = returns.std() * np.sqrt(bars_per_year) * 100
    downside_returns = returns[returns < 0]
    downside_vol = downside_returns.std() * np.sqrt(bars_per_year) * 100
    
    # Sharpe & Sortino
    sharpe = annual_return / volatility if volatility > 0 else 0
    sortino = annual_return / downside_vol if downside_vol > 0 else 0
    
    # Drawdown
    peak = equity_series.cummax()
    drawdown = (equity_series - peak) / peak
    max_drawdown = drawdown.min() * 100
    
    # Calmar ratio
    calmar = annual_return / abs(max_drawdown) if max_drawdown != 0 else 0
    
    # Trade stats
    winning_trades = [t for t in trades if t.pnl > 0]
    losing_trades = [t for t in trades if t.pnl < 0]
    win_rate = len(winning_trades) / len(trades) * 100 if trades else 0
    
    avg_win = np.mean([t.pnl for t in winning_trades]) if winning_trades else 0
    avg_loss = np.mean([t.pnl for t in losing_trades]) if losing_trades else 0
    profit_factor = abs(avg_win / avg_loss) if avg_loss != 0 else 0
    
    return {
        "total_return": round(total_return, 2),
        "annual_return": round(annual_return, 2),
        "volatility": round(volatility, 2),
        "sharpe_ratio": round(sharpe, 2),
        "sortino_ratio": round(sortino, 2),
        "max_drawdown": round(max_drawdown, 2),
        "calmar_ratio": round(calmar, 2),
        "win_rate": round(win_rate, 2),
        "total_trades": len(trades),
        "winning_trades": len(winning_trades),
        "losing_trades": len(losing_trades),
        "profit_factor": round(profit_factor, 2),
    }
```
